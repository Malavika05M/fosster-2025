# FOSSter(Web)

This repository contains the official website for the **FOSSter initiative**.

## Getting Started

### Installation

Clone the repository:

```bash
git clone https://github.com/amfoss/fosster-2025.git
cd fosster-2025
```
Install the dependencies
```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

### Run locally

Start the development server:

```bash
npm run dev
# or with Turbopack
npm run dev:turbo
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Production

```bash
npm run build
# or with Turbopack
npm run build:turbo
# or
yarn build
# or
pnpm build
# or
bun build
```
